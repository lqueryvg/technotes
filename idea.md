# IntelliJ IDEA – Mac Keyboard Shortcuts

| Shortcut | Action |
|---|---|
| `Cmd+Shift+O` | Find file by name |
| `Cmd+Shift+F` | Find in files |
| `Cmd+,` | Open settings |
| `Cmd+,` → Plugins tab | Open plugins |
| `Cmd+Shift+A` | Find action (command palette) |
| `Double Shift` | Search everywhere (files, actions, symbols, settings) |
