# Curl


```
-L                # follow redirects
-I                # headers only (--head)
-H "X-Foo: y"     # add a header
-X POST           # request type (--request)
-d data           # POST data
-o file           # output to file
-k                # for self-signed certs (--insecure)
-b name=val       # cookie
-b FILE           # cookie from file
-s                # silent
```
